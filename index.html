Welcome to the VigorXR Website!

VigorXR is a VR social multiplayer game, with a main focus on replayability, social features, and creation tools.
The development team is made up of 3 people:
LeonToGamer [Founder / Tool Developer]
Bubby932 [Core Developer & Head Artist]
Lizard [Support Team Head]

Remember, this site is currently being worked on, and should get updated a few times per day
If the website doesn't update for you when we announced a released update and or the website broke,
please try clearing your browser cache, in order for it to update properly.
Also, if you are on a phone, please turn it horizontally, as this website doesn't work great with
a vertical screen. We will remove this warning once the website works for a 9:16 aspect ratio.
