Patch 1.2.1

This is a very minor patch, so all changes are in one large list.

  • Improved performance in the login scene.
  • Fixed an issue where you could upload the same photo more than once.
  • Added printing photos.
  • Added local save options for both PNG and JPG versions of photos.
  • Redid handholds on A Helpful Hand level 1 and 2.
  • Improved performance on A Helpful Hand level 1 slightly.
